Setup Instructions

1
Download SumatraPDF: Download the official portable SumatraPDF executable from SumatraPDF Reader Website.
2
Place SumatraPDF: Copy the downloaded SumatraPDF.exe file and place it in the same folder where you run UCPS.exe (or add it to your Windows System PATH).
3
Run UCPS Client: Double-click UCPS.exe to open the interface. When prompted, log in with your registered shop operator credentials.
4
Start Spooling: Once authenticated, the desktop client automatically binds to your shop's printer and listens for jobs. All incoming student print jobs will compile and print automatically.
